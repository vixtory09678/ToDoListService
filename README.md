## Description

This project is powered by [Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Standalone Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Docker Compose (Service and Database)

```bash
$ docker compose up --build -d
```
